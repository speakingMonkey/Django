# Django博客系统
=========
##使用pythonweb框架django写的网站<br>
django版本1.9.6<br>
数据库:sqlite3<br>
前端：bootsharp<br>
具体文章访问个人博客:www.wanjiaxing.top<br>


版本1.0<br>
实现基本的博客功能。<br>
1文章的增加和删除。<br>
2文章列表和详情。 <br>
3评论功能以及后台管理.<br>
4分页功能<br>
